# Best Trades of June 2026

You can find all the [“Best trades of the month” articles since September 2025 on this link](https://cryptonarratives.substack.com/t/monthly-best-trades).

Just like the 5 past months, you are again getting access to the **full database** ***(link below)*** **of all interesting trades of June 2026**. *(I am progressively filling the database to include all trades from 2024 to now, and I’ll give you access to it whenever that’s ready. It will include several hundreds of examples of trades that you’ll be able to sort according to each field!)*

Each item of the database is one trade, and each item has these fields:

- Short description
- Ticker
- Month
- Type of coin
- Long or short
- Type of trade (this will be very useful in the future to filter all trades of a certain type over several months)
- Relevance tier: I rank these trades from 1 to 7 by importance/relevance
- Long description
- Primary pump: a checkbox used to mark the main pump for a given coin during the month. Filtering for checked boxes produces a clean, non-duplicated list of the month’s main pumps.
